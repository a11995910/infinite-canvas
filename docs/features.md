# 功能介绍

本文档记录当前项目已经实现的主要功能。

## 画布项目

- 支持创建多个画布项目。
- 支持项目重命名、删除、批量选择和批量删除。
- 支持单个画布项目导出为 JSON，也支持从 JSON 导入画布。
- 画布项目保存在浏览器本地，登录账号后暂不会自动同步到服务器。

## 无限画布

- 支持拖动画布、滚轮缩放、缩放滑杆和重置视图。
- 支持小地图定位，可开关小地图。
- 支持点阵、网格线、空白三种背景。
- 支持浅色和深色主题。
- 支持框选、多选、全选、取消选择、删除选中。
- 支持复制粘贴节点和节点之间的连线。
- 支持撤销和重做节点、连线、视口、背景和助手会话变化。
- 支持节点连线，并高亮当前节点相关的上下游节点和连线。
- 支持快捷键帮助，覆盖缩放、框选、全选、复制粘贴、撤销重做、删除、退出选择和拖入图片。
- 支持左上角“页面快捷罗盘导航”，点击罗盘（Compass）按钮即可在非侵入式的画布视口内，一键极速穿梭于主页、画布项目列表、生图工作台、视频创作台、提示词库和我的素材等核心页面。
- 画布顶部正式常态集成并展示全站置顶统一导航栏（AppTopNav），以便用户在全局业务流之间流畅同步。

## 节点

目前画布中有三类节点：

- 图片节点：展示上传图片、生成图片或素材库图片。
- 文本节点：保存提示词、说明文案、AI 文字回答等文本内容。
- 生成配置节点：汇总上游文本和图片，统一配置模型、比例、数量后批量生成图片或文本。

节点支持：

- 拖拽移动。
- 四角缩放。
- 图片节点等比缩放或自由比例切换。
- 查看节点基础信息和 JSON。
- 删除、复制、粘贴。
- 通过左右连接点建立上下游关系。

## 图片工作流

- 支持上传图片到新节点。
- 支持拖拽图片文件到画布。
- 支持替换已有图片节点内容。
- 支持下载图片节点。
- 支持把图片节点保存到“我的素材”。
- 支持图片裁剪，并把裁剪结果生成为新的图片节点。
- 支持本地多角度变换，并把结果生成为新的图片节点。
- 支持生成失败后重试。
- 批量生成多张图片时会先展示为图片组节点，支持叠卡预览、展开查看全部结果并设置主图。

## 生图工作台

生图工作台面向连续图片创作，支持：

- 侧边工作台和悬浮底部工作台两种布局，布局选择会保存在浏览器中。
- 提示词、参考图、参数分组折叠，折叠态显示当前摘要。
- 图片尺寸、质量、格式、压缩、审核、数量、模型和 API 模式配置。
- 多任务并发生成，每张结果卡片展示提示词、模型、接口、尺寸、质量、格式、耗时和失败信息。
- 生成中任务实时显示耗时，成功或失败后进入同一个历史结果网格。
- 历史结果支持全部展示和分类展示，支持新建分类、改名、删除分类、将结果加入分类。
- 结果卡片支持载入配置、重试、复制提示词、展开提示词、下载、加入参考图、加入“我的素材”和查看失败详情。
- 参考图来源会区分本地临时上传、生成结果、我的素材和素材库；删除参考图时只清理无用临时文件，素材和生成结果文件不会被误删。
- 支持直接从“我的素材”选择文本或图片；文本可写入提示词，图片可作为参考图且不会重复上传。

## 视频创作台

视频创作台面向连续视频创作，支持：

- 使用 Agnes-Video-V2.0 生成文本视频和参考图视频。
- 提示词、参考图、参数分组折叠，支持侧边工作台和底部工作台两种布局。
- 视频尺寸、清晰度、秒数、模型和并行任务数配置；Agnes 请求会按尺寸传递 `width`/`height`，按秒数换算 `num_frames` 和 `frame_rate`。
- 多任务并发生成，生成中、失败和历史结果统一进入“全部成果”卡片区；正在生成的任务支持通过接口轮询实时返回 Agnes-Video-V2.0 生成进度百分比（progress, 0~100%），并在生成卡片上配备具有 HSL 霓虹渐变与呼吸发光微动画效果的动态进度条，深度舒缓用户等待焦虑。
- 结果卡片展示提示词、模型、尺寸、清晰度、秒数、耗时、失败详情和视频来源。
- 视频来源区分 AI 临时 URL、云端存储和本地缓存；默认只记录上游返回的视频 URL，不自动下载占用本地存储。
- 支持把 AI 返回的视频 URL 手动同步到账号/R2 对象存储，同步成功后替换为云端 URL 并记录文件大小。
- 视频历史跟随账号同步；生成成功、删除记录和同步云端存储后都会更新账号视频历史。
- 历史结果支持载入配置、重试、复制提示词、展开提示词、下载到本地、保存到“我的素材”、批量删除和单条删除。
- 参考图支持本地上传、剪切板、“我的素材”和素材库选择；临时参考图只有在不被生成任务或历史记录占用时才会真删除。

## AI 生成

项目支持本地直连和云端渠道两种模型调用方式：

- `/v1/images/generations`：文生图。
- `/v1/images/edits`：图生图/参考图编辑。
- `/v1/chat/completions`：文本问答和带图问答。
- `/v1/videos`：视频任务创建。
- `/v1/videos/{id}`：视频任务轮询和结果读取。
- `/v1/models`：读取模型列表。

可配置项：

- Base URL。
- API Key。
- 本地多个渠道，每个渠道可配置独立模型列表。
- 云端多个后台渠道，前端可按图片、视频、文本分别选择渠道和模型。
- 默认图片模型、视频模型、文本模型。
- 前端生图统一使用 Images API。
- 流式传输、中间步骤图数量、返回 Base64、Codex CLI 兼容模式和请求超时。
- 图片质量。
- 图片比例。
- 输出格式：PNG、JPEG、WebP。
- 压缩比例。
- 审核级别。
- 生成数量。
- 视频尺寸、清晰度、秒数和并行任务数。

普通图片/文本节点可以直接输入提示词生成结果。生成配置节点可以读取上游节点内容，并按节点自己的配置批量生成多个图片或文本结果。生成配置节点支持预览当前提示词和参考图输入，并调整输入顺序。

## 创作工作流

创作工作流用于把一类可复用的创作需求沉淀成模板，例如电商海报、小红书封面、菜谱步骤图或写真风格图。

- 支持个人工作流和公开工作流。
- 公开工作流对所有用户可见，其他用户可复制成个人副本后编辑。
- 工作流变量支持短文本、长文本、数字、选项和开关。
- 选项变量支持用 `/` 或换行分隔多个候选项。
- 开关变量运行时会渲染为“开启/关闭”，适合写入“是否显示价格”“是否保留背景”等提示词片段。
- 工作流可配置图片模型、API 模式、尺寸、质量、格式、数量、流式、Base64、Codex CLI 和超时。
- 运行工作流时可以上传参考图或从“我的素材”选择图片作为参考。
- 工作流运行结果会写入生图工作台历史，并自动归入工作流同名分类。
- AI 创建工作流支持输入需求和参考图，由文本模型生成结构化工作流草稿；Agent 弹窗可单独选择不同渠道和文本模型。
- 多图生成工作流支持先调用文本模型生成系列图片提示词草稿，再逐条生成或按并发上限批量生成。
- 多图工作流可独立配置“提示词生成模型/渠道”和“图片生成模型/渠道”，适合用低成本文本模型拆分方案，再用指定图片模型出图。
- 多图草稿支持审核、编辑、复制、上下移动、删除、批量追加统一要求、失败重试和刷新后恢复。
- 工作流列表支持按分类筛选，并内置“小红书文章配图组”多图模板。

## 画布助手

画布右侧助手面板支持：

- 文本问答。
- 生图。
- 读取当前选中节点作为引用。
- 自动把选中节点的上游节点也纳入引用。
- 粘贴图片到助手输入框并插入画布。
- 历史会话。
- 删除单条或多条会话。
- 重试回答。
- 把助手生成的文本插入画布。
- 把助手生成的图片插入画布。
- 折叠和展开助手面板。
- 从“我的素材”插入文本到输入框，或插入图片作为 AI 生图参考图。

## 提示词库

前台提示词库支持：

- 按标题搜索。
- 按标签筛选。
- 按来源筛选。
- 查看提示词详情。
- 查看封面和结果图。
- 复制提示词。
- 把提示词加入“我的素材”。

后台提示词管理支持：

- 查询提示词。
- 新增、编辑、删除提示词。
- 按分组和标签筛选。
- 查看远程提示词源。
- 同步内置远程提示词源。

当前内置远程源包括多个 GPT Image / GPT-4o / Nano Banana Pro 相关提示词仓库。

## 素材

“我的素材”是浏览器本地素材库，支持：

- 新增文本素材和图片素材。
- 编辑素材标题、封面、标签、来源、备注和内容。
- 删除素材，并引入全系统多级 GC 引用扫描保护：自动扫描其余资产与画布项目节点是否仍在引用该资源。若判定为全站无引用，将自动联动物理删除底层 IndexedDB 缓存和云端/服务器存储文件（完美支持图片与 server: 云端视频资源），确保资产引用安全与存储空间零浪费。
- 按关键词搜索。
- 按类型筛选。
- 分页浏览。
- 复制文本素材。
- 下载图片素材。
- 从提示词库、画布节点和服务器素材库加入素材。
- 在画布、生图工作台、视频创作台、画布 AI、工作流运行弹窗和 AI 创建工作流弹窗中插入素材。
- 图片素材会按当前存储模式写入 IndexedDB 或服务端 SQLite + S3/R2；作为参考图使用时不会重复上传。

“素材库”是服务器素材库，支持：

- 按标题搜索。
- 按类型筛选。
- 按标签筛选。
- 查看素材详情。
- 复制文本或图片链接。
- 加入“我的素材”。
- 在画布中插入素材。

后台素材库管理支持：

- 查询素材。
- 新增、编辑、删除素材。
- 按类型和标签筛选。

## 账号和后台

- 注册功能暂时关闭。
- 仅允许管理员账号登录。
- 支持 JWT 会话。
- `/api/auth/me` 可读取当前用户，未登录时返回访客用户。
- 首次启动时可根据环境变量创建默认管理员。
- 管理员后台目前包含提示词管理和素材库管理。
- 后端已有用户管理接口，但前端暂未实现用户管理页面。

## 后端能力

- Gin 提供 API 服务。
- Docker 运行时由 Next.js 提供页面入口，`/api/*` 请求代理到内部 Go 服务。
- GORM 管理数据库连接和自动迁移。
- 支持 SQLite、MySQL、PostgreSQL。
- 数据库保存用户、提示词分组、提示词、服务器素材、用户配置、工作流、对象存储文件索引和算力点日志。
- 支持 SQLite + S3/R2 存储模式，图片文件可保存到 Cloudflare R2 或其他 S3 兼容对象存储。
- 视频创作台可把 AI 返回的临时视频 URL 手动同步到 Cloudflare R2 或其他 S3 兼容对象存储。
- 支持管理员配置系统对象存储，也支持在管理员允许后由用户配置自己的 S3/R2 存储。
- 业务接口统一返回 `{ code, data, msg }`。

## 当前限制

- “我的素材”仍以浏览器本地素材库为主，不会完整随账号同步；其中图片文件会按存储模式写入 IndexedDB 或对象存储。
- 本地直连模式的 API Key 保存在浏览器本地；云端渠道由后端代理并由管理员统一配置密钥。
- 服务器素材库目前主要保存 URL 或文本；“我的素材”弹窗新增素材走用户侧存储流程。
- 画布更适合桌面端使用，移动端触控体验还未系统完善。
