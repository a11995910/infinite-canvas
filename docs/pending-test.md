# 待测试

- Sub2API 自定义菜单 iframe 嵌入模式下，页面会读取 `ui_mode=embedded`、`token` 和 `src_host`，通过 Sub2API `/api/v1/auth/me` 校验当前用户并创建或复用画布本地会话，避免 iframe 内二次登录；登录成功后再通过服务端代理获取当前用户 API Key，并自动写入本地直连渠道。嵌入模式下的画布列表、项目详情、顶部导航、移动导航和登录页兜底都会保留 Sub2API 嵌入参数；如果换票失败，只显示 Sub2API 登录状态错误，不再跳转或展示画布自己的账号密码登录页。AI 请求会经 `/api/sub2api/proxy/*` 由画布服务端转发到 Sub2API，避免浏览器跨域直连。画布服务端生成代理地址时会优先使用反向代理传入的公开协议和域名，适配 HTTPS iframe 场景。
- Sub2API 嵌入模式下的配置弹窗改为简化视图，只展示当前账号可用 Key 选择，以及图片模型、文本模型和视频模型预留选择；不会再要求用户填写渠道名称、Base URL、API Key 等直连配置。切换 Key 后会优先通过 Sub2API 代理刷新模型列表，失败时保留默认模型兜底。
- 生图配置已禁用 Responses 图像模式；配置弹窗、画布图片节点、生成配置节点、生图工作台和工作流编辑器不再展示图片协议选择，旧配置和旧历史记录会按 Images 模式处理，真正的文本对话模式不受影响。
- 生图参数新增输出格式、压缩比例和审核级别；画布节点、配置节点和助手生图会随请求传递 `output_format` 等参数。
- 生图工作台支持连续提交多个并发生图任务，结果卡片展示任务时间、提示词、模型、尺寸、质量、格式和耗时，失败任务展示上游报错并支持单任务重试。
- 生图工作台将生成记录并入生成结果区展示，结果卡片支持提示词展开和复制；生成中卡片实时显示耗时，参考图区域支持展开/收起且收起时保留已选参考预览。
- 生图记录保存生成接口返回的内联图片作为本地存储回退，避免离开生图工作台后历史结果图片无法显示。
- 生图工作台新增侧边/底部工作台切换并持久化选择；提示词、参考图、参数均支持展开/收起摘要展示，历史结果卡片改为紧凑网格。
- 无限画布图片节点支持双击查看大图；生成中的图片节点会显示实时耗时，生成完成或失败后记录本次耗时。
- 前端云端渠道配置会展示后台渠道摘要；当公开可用模型未手动勾选时，会回退使用已启用私有渠道的模型列表，避免云端模式无模型可选。
- 画布项目导出改为下载 `.zip` 压缩包，包内包含 `projects.json` 和当前画布引用到的本地图片、视频文件，避免只导出 JSON 时丢失媒体内容。
- 画布库支持多选后一键导出多个画布项目，导出的压缩包可一次恢复多个项目。
- 画布项目导入改为读取新版 `.zip` 压缩包，会先按 `projects.json` 中的文件映射恢复图片、视频到本地存储，再插入画布项目，导入成功后仍停留在画布库。
- 修复删除画布图片节点或清空画布后撤销时，节点信息恢复但本地图片数据已被清理导致图片丢失的问题。
- “我的素材”类型筛选区右侧新增文本样式的导出素材和导入素材入口，可将全部素材导出为包含 `assets.json` 与图片、视频文件的压缩包，并从压缩包恢复素材。
- 未登录状态下，画布右上角不再显示用户头像菜单、用户名称、算力点余额和退出登录入口，改为显示登录入口；快捷键入口仍可直接打开。
- 生图工作台的图片参数区复用画布里的紧凑版图像设置面板，尺寸、质量、生成张数的交互保持一致；工作台仍保留独立的模型选择。
- 生图工作台新增生成记录配置持久化：每次生成会保存提示词、参考图、模型、质量、尺寸和张数，结果图写入本地图片存储后记录只保存 `storageKey`；点击历史记录会回填本次生成配置并预览结果。
- 视频设置抽成画布和视频创作台共用的紧凑面板，清晰度、尺寸、秒数按固定网格选择并支持手动输入；尺寸选择 `auto` 时请求不传 `size`。
- 修复画布和生图工作台选择图片尺寸后，请求图片生成/编辑接口未携带 `size` 参数的问题；`auto` 不传，其余比例或像素尺寸会随请求发送。
- 修复生图工作台和画布生图请求中 `quality` 参数可能传入上游不支持值导致 400 的问题；请求前会归一化质量枚举，`auto` 或异常值不再发送给上游。
- 管理后台新增/编辑渠道时，渠道可用模型支持通过弹窗按“新获取、已有”分组选择，并可在弹窗内手动增加模型或拉取模型列表后再写回表单。
- 管理后台编辑渠道时，API Key 留空不再触发必填校验，表示沿用已保存的密钥；新增渠道仍要求填写 API Key。
- 管理后台公开配置里的系统可用模型候选项改为由已启用渠道中选择的模型合并去重生成，最终开放哪些模型仍由公开配置里手动勾选。
- 视频生成请求参数使用 `resolution_name`、`preset=normal`、`input_reference[]`，支持清晰度、尺寸、秒数快捷选择和手动输入，并支持最多 7 张参考图。
- 视频生成接口优先使用 `Agnes-Video-V2.0` 模型；Agnes 视频请求对齐官方参数，文本和参考图生成使用 `num_frames`、`frame_rate`、`width`、`height`，视频时长会换算为满足 `8n + 1` 且不超过 441 的帧数，单张参考图使用 `image`，多张参考图使用 `extra_body.image`，参考图优先传可公开访问的图片 URL；轮询完成后会从 `video_url` 或 `remixed_from_video_id` 读取视频地址并直接作为结果展示，不再自动下载并写入本地媒体存储。
- 视频生成结果支持下载到本地文件；结果卡片显示 AI 临时 URL、云端存储或本地缓存来源，生成成功、删除记录和同步云端存储后会把视频历史同步到账号，成功后替换为云端存储 URL 并显示同步后的文件大小。
- 视频创作台历史结果卡片的选择框旁新增单条删除按钮，可直接删除单个视频生成结果。
- 视频生成前端补充本地直连 AI 调用日志，一次生成只记录最终成功或失败日志，包含请求摘要、最终任务响应、视频地址和错误信息；参考图等大字段会在日志中脱敏。
- 视频创作台升级为“全部成果”卡片区，生成中任务、失败任务和历史成果统一展示，并支持多任务并行提交、侧边/底部工作台切换、提示词复制、重试、下载和保存到素材。
- 视频创作台失败卡片和历史失败记录新增详情弹窗，后端代理会尽量透出上游 `error.message` / `msg`，便于查看 Agnes 接口返回的具体错误。
- 视频创作台参考图支持从“我的素材/素材库”选择和本地上传；临时上传/剪切板参考图仅在未被生成任务或历史记录占用时真删除，被占用时只从当前工作台移除。
- 画布视频设置浮层改为挂载到页面根层级并使用自建浮层交互，避免被节点悬浮工具栏遮挡或点击面板内容时关闭。
- 画布生成配置节点的生图参数改为复用图像设置浮层，支持在同一个入口里调整质量、尺寸和生成张数。
- 视频清晰度输入框改为只输入数字，提交请求时再拼接 `p` 单位。
- 视频生成前端会识别后端 `{ code, msg }` 错误响应，创建失败不再继续轮询 `undefined`。
- 新增 `/video` 视频创作台页面，参考生图工作台布局，支持提示词、参考图、视频参数、生成结果、保存素材、下载和本地生成记录；清晰度和秒数均支持常用值选择与手动输入，生成记录只保存媒体 `storageKey` 并可回填本次提示词、参考图和参数。
- 视频创作台生成前会把模型、尺寸、秒数、清晰度归一化为视频接口支持的参数，并展示后端返回的错误信息，避免页面侧残留的生图参数影响视频请求。
- 创作工作流支持一排 3 个模板卡片、右侧抽屉嵌入生图工作台、个人/公开模板、复制公开模板、运行多工作流并发任务和结果写入生图历史。
- 工作流 AI 创建弹窗支持输入参考图、选择“我的素材”作为参考、独立选择不同渠道的文本模型，并在生成草稿后应用到工作流编辑器。
- 工作流变量支持短文本、长文本、数字、选项和开关；选项编辑框支持 `/` 分隔，开关变量运行时会渲染为“开启/关闭”。
- 生图工作台、画布 AI、工作流运行弹窗和工作流 AI 创建弹窗都支持从“我的素材”选择文本/图片，图片复用已有 `storageKey`，不会重复上传。
- 参考图删除区分临时上传和已有素材/生成结果：临时未使用文件会删除 R2/SQLite，素材和生成结果引用只从当前工作台移除。
- 新增 SQLite + S3/R2 文件存储、用户自定义 R2、管理员 R2 容量统计、账号同步模型配置/存储配置/画布/生图历史等能力。
- 项目文档新增二开功能说明、自定义前后端接口、源码部署和 GitHub 开源协作文件。
