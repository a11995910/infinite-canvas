# CHANGELOG

## Unreleased

## v0.3.3 - 2026-06-09

+ [新增] 画布图片节点新增迭代面板，支持生成变体、换风格、换背景、重构图和局部重绘，并保留参考图、模型参数、提示词和生成耗时。
+ [新增] 图片迭代支持参考图角色标注，可按主体、风格、构图、色彩、背景和锁定等用途影响后续图生图提示。
+ [优化] 图片迭代局部重绘会向图片编辑接口传递 mask，支持模型优先约束选区修改。
+ [修复] 修复图片文件存入本地 IndexedDB 后可能因原始 File 句柄失效导致刷新后参考图丢失的问题。
+ [修复] 修复画布多个图片迭代任务共享运行状态，导致一张图片迭代中另一张图片误显示“生成中”的问题。

## v0.3.2 - 2026-06-02

+ [新增] 画布（Infinite Canvas）页面左上角集成“页面罗盘导航”菜单，采用 Compass 罗盘图标和 Dropdown 交互，支持在画布视口中一键流畅穿梭至系统首页、画布项目列表、生图工作台、视频创作台、提示词库和我的素材。
+ [新增] 视频创作台全面打通 `Agnes-Video-V2.0` 生成进度，视频卡片实时显示 "正在创作 X%" 文本，并配备了具有 HSL 霓虹渐变、轻微拉伸与脉冲发光动效的高级呼吸条形进度条 UI。
+ [新增] 素材库删除资产触发全系统 GC 引用扫描保护。在删除素材资产时，自动异步比对其余素材和画布节点的引用。若判定为“全站无其他地方引用”，将同时自动联动物理删除本地 IndexedDB 文件与云端服务器文件（包括图片和云端视频）。
+ [新增] 视频文件存储物理删除支持：在 `file-storage.ts` 中新增对 `server:` 云端视频文件的物理擦除，彻底杜绝云端和服务器存储的垃圾文件残留。
+ [优化] 画布（Infinite Canvas）正式常态释放展示置顶统一导航栏，方便项目统一流转。
+ [优化] 精简并重构生图/视频工作台布局。解决生图工作台双“开始创作”按钮重复渲染问题；精简视频工作台，重构底部快速控制网格为完美对称布局。

## v0.3.1 - 2026-06-01

+ [新增] 视频创作台升级为全部成果卡片区，支持生成中、失败和历史结果统一展示，多任务并行、侧边/底部工作台切换、提示词复制、重试、下载和保存素材。
+ [新增] 视频生成结果支持 AI 临时 URL 与云端存储状态标识，支持手动同步到 R2/对象存储，成功后用云端 URL 替换并同步账号视频历史。
+ [新增] 视频创作台支持从“我的素材/素材库”和本地上传参考图，临时参考图按是否被任务或历史记录占用决定真删除或仅移除。
+ [新增] 视频历史结果卡片新增单条删除按钮，批量删除和单条删除都会同步更新账号视频历史。
+ [优化] Agnes-Video-V2.0 视频请求对齐官方参数，按工作台尺寸传递 `width`/`height`，按秒数换算满足 `8n + 1` 且不超过 441 的 `num_frames`，并传递 `frame_rate`。
+ [优化] 视频结果不再自动下载并写入本地媒体存储，直接记录上游返回的视频 URL；用户点击下载时才保存到本地文件。
+ [优化] 画布视频节点复用视频生成返回 URL，避免自动写入本地大体积视频；视频生成错误会透出上游详细信息。
+ [优化] 视频 AI 调用日志补充最终任务响应、视频地址和错误信息，参考图等大字段会脱敏。
+ [修复] 修复 Agnes 参考图/多参考图请求字段错误、完成后卡片仍停留生成中、成功视频无法播放和竖屏尺寸未按设置传递的问题。
+ [文档] 更新视频创作台、账号同步、数据库和对象存储相关文档。

## v0.3.0 - 2026-05-31

+ [新增] 创作工作流支持“多图生成工作流”，可先用文本模型生成系列图片提示词草稿，再逐条或批量并行生成图片。
+ [新增] 多图工作流支持独立配置提示词文本模型/渠道、图片生成模型/渠道、目标张数、并发上限、审核开关和系列拆分规则。
+ [新增] 工作流列表支持分类筛选，并内置“小红书文章配图组”等多图模板；AI 创建工作流可识别多图/系列/组图意图并生成对应配置。
+ [优化] 多图提示词草稿支持持久化、编辑、复制、删除、上下移动、批量追加要求和失败重试，结果继续写入生图历史并自动按工作流归类。
+ [优化] AI 调用日志页面将详情拆分为“请求详情”和“返回详情”，完整展示请求/返回内容，并优化本地直连日志开关和顶部工具栏排版。
+ [修复] 修复 AI 日志 Base64 清洗正则在 Go 下的启动 panic，并增强失败日志噪声清理，方便定位真实接口错误。
+ [修复] 修复多处标签列表重复 key 风险，避免重复标签导致 React 渲染告警。
+ [文档] 新增多图生成工作流开发文档和 v0.3.0 审查修复记录。

## v0.2.0 - 2026-05-31

+ [新增] 生图渠道支持 Images API 流式传输、Base64 返回、Codex CLI 兼容模式、请求超时和输出格式配置。
+ [优化] 生图工作台支持多任务并发、任务参数展示和失败错误展示，云端渠道前端展示后台渠道摘要并自动补齐可用模型。
+ [优化] 生图工作台将历史记录并入结果区，支持提示词展开复制、生成中耗时统计、参考图折叠和结果图片持久显示回退。
+ [优化] 生图工作台新增侧边/底部工作台切换、三段折叠摘要和紧凑历史卡片；画布图片节点支持双击查看大图和生成耗时显示。
+ [新增] 创作工作流支持个人/公开模板、AI 创建工作流、工作流运行结果入库、素材参考图复用和独立选择 Agent 文本模型渠道。
+ [新增] 支持 SQLite + S3/R2 文件存储、用户自定义 R2、存储容量统计、账号同步模型配置/存储配置/画布/生图历史。
+ [新增] AI 调用日志改为本地按天 JSONL 文件，后台支持查询详情、手动清理和自动保留策略。
+ [优化] 生图结果下载会读取图片 Blob 后直接保存到本地，避免跨域 URL 被浏览器打开。
+ [优化] 生图/视频/文本/工作流/Agent 系统提示词集中到后台配置，可随时调整。
+ [文档] 完善二开部署、接口代理、自定义前后端地址、贡献指南、安全说明、Issue/PR 模板和 CI。
